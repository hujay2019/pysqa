# Command Line Interface
```
python -m pysqa --help
```

* `-f`, `--config_directory`
* `-p`, `--submit`
* `-q`, `--queue`
* `-j`, `--job_name`
* `-w`, `--working_directory`
* `-n`, `--cores`
* `-m`, `--memory`
* `-t`, `--run_time`
* `-b`, `--dependency`
* `-c`, `--command`
* `-r`, `--reservation`
* `-i`, `--id`
* `-d`, `--delete`
* `-s`, `--status`
* `-l`, `--list`
* `-h`, `--help`
